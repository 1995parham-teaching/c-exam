\قسمت{فیبوناچی در فایل}

تابع زیرا به شکلی تکمیل کنید
که جمله‌ی \متن‌لاتین{\متن‌ایتالیک{n}}ام فیبوناچی را محاسبه کرده
و آن را در \متن‌سیاه{انتهای} فایل داده شده یادداشت کند.
در نظر داشته باشید عدد به صورت متنی نوشته می‌شود.

\begin{latin}
\begin{minted}{c}
void fib_in_file(char *name, int n) {
    // your code goes here
}
\end{minted}
\end{latin}

برای درک بهتر از عملکرد تابع به مثال‌های زیر توجه کنید:

\begin{latin}
\begin{minted}{output}
    Input: 1
    Output: 1
\end{minted}

\begin{minted}{output}
    Input: 2
    Output: 1
\end{minted}

\begin{minted}{output}
    Input: 3
    Output: 2
\end{minted}
\end{latin}

۴ نمره

در نظر داشته باشید که حتما در هنگام باز کردن فایل خطا را نیز مورد بررسی قرار دهید.
