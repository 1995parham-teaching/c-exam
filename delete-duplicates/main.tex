\قسمت{پاک کردن تکراری‌ها}

تابع زیر را به شکلی تکمیل کنید که اعضای تکراری از لیست پیوندی داده شده را، حذف نماید.

\begin{latin}
\begin{minted}[bgcolor=LightGray]{c}
struct node {
    int value;
    struct node *next;
};

void delete_duplicates(struct node *head) {
    // your code goes here
}
\end{minted}
\end{latin}

برای درک بهتر از عملکرد تابع به مثال‌های زیر توجه کنید:

\begin{latin}
\begin{minted}{output}
Input: 1->1->2
Output: 1->2
\end{minted}

\begin{minted}{output}
Input: 1->1->2->3->3
Output: 1->2->3
\end{minted}
\end{latin}
