\section{پاک کردن تکراری‌ها}
\paragraph{}
تابع زیر را به شکلی تکمیل کنید که اعضای تکراری از لیست پیوندی داده شده را حذف نماید.

\begin{latin}
\begin{lstlisting}[language=c]
    struct node {
        int val;
        struct node *next;
    };

    void delete_duplicates(struct node *head) {
        // your code goes here
    }
\end{lstlisting}
\end{latin}

برای درک بهتر از عملکرد تابع به مثال‌های زیر توجه کنید:

\begin{latin}
\begin{verbatim}
    Input: 1->1->2
    Output: 1->2
\end{verbatim}

\begin{verbatim}
    Input: 1->1->2->3->3
    Output: 1->2->3
\end{verbatim}
\end{latin}
